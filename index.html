What is a Computer?

Hardware & Software

Hello and welcome! This class is a fun way to learn about computers in English. We will talk about two big parts of a computer: hardware and software. Let's begin!

Hardware: You Can Touch It

Hardware is the physical part of a computer. You can hold it, see it, and touch it. The mouse, keyboard, and monitor are all hardware. The parts inside, like the CPU (the computer's brain) and motherboard (the main board), are also hardware.

Think of hardware like the body of a robot!

Software: The Programs

Software is the programs and data that tell the hardware what to do. You cannot touch software. For example, your favorite game, a web browser, or the Operating System (OS) like Windows or macOS are all software. The OS is the most important program because it helps all other hardware and software to work together.

Think of software as the robot's brain and thoughts!
